# DevApply Frontend

React frontend for the DevApply job application automation system.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Features

- User registration
- Resume upload with drag & drop
- Modern, responsive UI
- Error handling and validation

## Backend Connection

Make sure the FastAPI backend is running on `http://localhost:8000` before starting the frontend.

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
